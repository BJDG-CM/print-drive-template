    신뢰 기기 전용

    암호화 업데이트 패키지 만들기

    전체 파일 비밀번호를 다루는 관리 작업입니다. 본인만 사용하는 신뢰 기기에서 진행하세요.

    추가하거나 교체할 파일 선택 파일을 선택하면 암호화 업데이트 패키지를 준비합니다.
    다음 단계

    다운로드한 패키지를 로컬에서 npm run update:check -- <package.zip>로 검사한 뒤 npm run update:apply -- <package.zip>로 적용하세요. 패키지를 만드는 것만으로 공개 파일은 바뀌지 않습니다.

    소유자 전용 관리
